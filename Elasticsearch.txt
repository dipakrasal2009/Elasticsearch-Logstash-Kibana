ğŸŸ¨ Search Operations
------------------------------------------------------------------------------------------------
13. Match All Documents
GET /students/_search
{
  "query": {
    "match_all": {}
  }
}

14. Search by a Field
GET /students/_search
{
  "query": {
    "match": {
      "city": "Pune"
    }
  }
}

15. Search Using Query Parameter
GET /students/_search?q=city:Pune

16. Filter Search (Exact match, no scoring)
GET /students/_search
{
  "query": {
    "term": {
      "age": "22"
    }
  }
}
